## 2. Mission & Positioning

Europe's first fully DSGVO-compliant AI infrastructure platform. Unifies all leading AI models (ChatGPT, Claude, Gemini, etc.) into one platform with a single account for the entire team, no per-user fees, and full data sovereignty on German servers.

**Key pillars:** All-in-One Platform, DSGVO/GDPR-Compliant (Frankfurt servers), No-Code Editor (SiteCreator), Enterprise-Ready (EU AI Act, automated AVV), European Alternative to US platforms.

## 3. Product Architecture

### 3.1 Assistant Types

| Type                           | Description                                                         |
| ------------------------------ | ------------------------------------------------------------------- |
| xChat                          | Central AI chat, auto-selects optimal model per query               |
| Chat Assistants                | Custom AI assistants, no coding required                            |
| Form Assistants                | Structured task assistants with guided inputs                       |
| Phone Assistants               | AI voice agents for inbound calls (Twilio + ElevenLabs/Deepgram)    |
| Workflow Assistants (SiteFlow) | Complex process automation via n8n, wrapped in simple dashboard UIs |

### 3.2 Core Technologies

| Technology  | Purpose                                                            |
| ----------- | ------------------------------------------------------------------ |
| SiteCore    | Proprietary knowledge management — hallucination-free AI results   |
| DMSP        | Dynamic Multi Step Prompting — multi-model orchestration           |
| SiteCreator | No-code AI assistant builder                                       |
| SiteFlow    | Workflow automation layer wrapping n8n in user-friendly dashboards |

### 3.3 Integrated AI Models

ChatGPT (OpenAI), Claude / Claude Opus / Claude Sonnet (Anthropic), Gemini (Google), Meta (LLaMA), Mistral, ElevenLabs (voice), Twilio (telephony), Deepgram (STT).
